### GitHub Copilot features

**Copilot for chat** - brings a ChatGPT-like chat interface to the editor, natively integrates with VS Code and Visual Studio. Recognizes the code and what error messages appear. 

**Copilot for pull requests** - AI-powered tags in pull request descriptions that organization admins and repository owners can install. It automatically fills out these tags based on the changed code, which then can be reviewed and modified. 

**Copilot for the CLI** - can compose commands and loops, and it can throw obscure find flags to satisfy your query.

### Subscription plans

**GitHub Copilot Free** - includes 2000 code completions per month, 50 chat requests per month, and access to GPT-4o and Claude 3.5 Sonnet models. 
	Key features:
		- Code completions in supported editors
		- Limited monthly completions and chat requests 
		- Access to advanced AI models

**GitHub Copilot Pro** - designed for individual developers who want enhanced capabilities beyond the Free plan
	Key features:
		- Unlimited code completions and chat requests
		- Priority access to the latest AI models
		- Advanced code suggestions and explanations
		- Integration with supported IDEs 
		- Automated test generation and code explanations features

**GitHub Copilot Pro+** - includes all the features of Copilot Pro with additional access.
	Key features:
		- All Pro features
		- Additional request capacity
		- Priority infrastructure access

**GitHub Copilot Business** - for organizations that want to manage Copilot access and benefit from additional security and compliance features.
	Key features:
		- All Pro and Pro+ features
		- Centralized management and policy controls
		- Security vulnerability filtering
		- Code referencing and public code filtering
		- IP indemnity and enterprise-grade security, safety and privacy
		- Chat in IDE and mobile
		- Filter for public code

**GitHub Copilot Enterprise** - designed for large organizations that require advanced capabilities.
	Key features:
		- All Business features
		- Personalized code suggestions 
		- Integration with GitHub Enterprise Cloud
		- AI-powered search and documentation generation across the codebase
		- Enhanced pull request support with AI-powered tags and summaries
		- Organization-wide customization and fine tuning of Copilot models
		- Deep integration with GitHub for chat
