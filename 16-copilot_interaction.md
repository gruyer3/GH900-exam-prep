**Inline suggestions** - as you type, Copilot analyzes your code and context to offer real-time code completions. To accept a suggestion, select Tab key or the > (right arrow key), to reject a suggestion, keep typing or select the Esc key.

**Command palette** - provides quick access to the various functions in Copilot:
- Open the command palette with Ctrl+Shift+P
- Enter Copilot to see available commands
- Select actions like Explain This or Generate Unit Tests to get assistance

**Copilot chat** - interactive feature that enables communication by using natural language. 

**Inline chat** - enables context-specific conversations directly within code editor.
- Place your cursor where you want assistance
- Use Ctrl+I 
- Ask questions or request changes specific to that code location
Slash commands are shortcuts that allow you to quickly perform actions. Some common slash commands include:
- `/explain` - provides an explanation of selected code
- `/suggest` - offers code suggestions
- `/tests` - generates unit tests 
- `/comment` - converts comments into code snippets
